# Language-Translator
“A Python-based GUI application that translates English text into multiple languages using Hugging Face Transformers. Built with Tkinter, it offers an intuitive interface and integrates text-to-speech functionality using pyttsx3 for audio output, enhancing accessibility and user experience.”

## 🚀 Features
- GUI using Tkinter
- Multi-language translation
- Text-to-Speech output
- Easy-to-use interface

## 🛠️ Technologies Used
- Python
- Transformers (Hugging Face)
- Tkinter
- pyttsx3

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python translator_gui.py
